# Code Review Rules

## JavaScript & Development
- Use modern ES6+ features.
- Prefer `const` and `let` over `var`.
- Use descriptive naming conventions for variables and functions.
- Keep functions small, focused, and single-purpose.

## HTML & CSS Layouts
- Write semantic HTML5 tags (e.g., `<header>`, `<main>`, `<section>`, `<nav>`, `<footer>`).
- Organize CSS styles using custom properties (variables) for design tokens.
- Design responsively using CSS Flexbox and Grid.
- Implement glassmorphism using semi-transparent borders, backdrop filters, and soft shadows.

## Animations & Scroll
- Synchronize animations cleanly with GSAP and Lenis scroll controllers.
- Monitor repaint costs to keep scroll performance >= 50 FPS.
